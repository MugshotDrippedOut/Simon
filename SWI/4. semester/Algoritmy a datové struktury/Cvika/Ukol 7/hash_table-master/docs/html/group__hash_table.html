◆ HashTable_Init()

bool HashTable_Init ( HashTable *  table,
size_t  size,
bool  takeOwnership 
)

Initializes the hash table.

It creates an array of pointers to items of the table (the HashTable::buckets), sets the HashTable::size to size and the HashTable::size to 0, and sets every bucket pointer to NULL.

Parameters
[out]tablePointer at initialized hash-table.
[in]sizeOur desired size of table
[in]takeOwnershipIf the argument takeOwnership is true, the value of item will be deleted as well
Returns
Returns true if table was initialized, otherwise it returns false.
