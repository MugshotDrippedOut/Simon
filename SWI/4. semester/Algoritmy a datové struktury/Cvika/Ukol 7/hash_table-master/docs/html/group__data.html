Detailed Description

Abstract data type that defines interface for all derived types. It defines basic data structure that contains table of virtual functions and the pointer to the data that are associated with the given instance. Thanks to this we can store the pointers in a custom data structure and use the provided API to call the derived functions without further distinguishing the used type.
